# Spring MVC Film CRUD Application

## Description

An Application allows a user to search for film records by an Id or keyword from a database. Additionally, a user may add, update, and delete films from a database.




## Technology Used 

* Spring MVC
* Spring ToolSuite
* MySQL
* Gradle
* Java 8
* HTML
* Bootstrap
* JSP tag library
* Github
* CSS
* Tomcat Server
